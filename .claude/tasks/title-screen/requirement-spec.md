# Requirement Spec for Title Screen
The title screen is the landing page of the game.
## Functional Requirement
1. The title screen should show title.png
2. The title screen should show the game name. see .github/tasks/game-design/
3. There are three options in the title screen: New Game, Continue, and Settings.
4. For this task, only New Game should work, it takes the flow to level 0 (序章)
5. Play title.mp3 when rendering title screen.
