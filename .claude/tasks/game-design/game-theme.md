# Game Theme
The game is a single player texture adventure game with the background in Three Kingdoms age in ancient China.
The game features rich, imaginative and engaging narration, as well as strategic gameplay.

1. The name of the game is "三国志重编：刘皇叔的命运分岔"
2. The game starts from June 194, when Liu Bei deployed his troop in Xiao Pei.
3. The game language is Chinese.
